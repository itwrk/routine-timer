☁️ 端末間同期（スマホ⇄Mac）

付属の rvt-sync.gs を自分のGoogleアカウントのApps Scriptにデプロイして、URLと合言葉を両端末に入れるだけ。データはあなたのGoogle Drive内に保存されます。AIのAPIキーは同期されません（端末ごとに保持）。

📅 Googleカレンダー

ホームの「今後の予定」表示もこの連携を使います。GASを最新版（rvt-sync.gs v2）に更新し、「デプロイを管理→新バージョン」で反映してください。

📓 Notion連携（GAS Webhook）

完了ごとに date / time / routine / done / total / skips / durationMin をJSONでPOSTします。筋トレログと同様、GAS側でNotion APIに中継してください。GASの `doPost(e)` で `JSON.parse(e.postData.contents)` を受けてNotionページ作成、でOK。

💾 データ
